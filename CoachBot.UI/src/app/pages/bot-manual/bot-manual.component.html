Bot Manual

CoachBot is the community's matchmaking Discord Bot, which enables easy matchmaking between teams, in multiple regions across the world. To use CoachBot for your team, create a team via the Hub, and navigate to the "Discord" tab of the team editor.

Lineup Management
Name Command Example Description
View Lineup !list !list Returns the current channel's lineup
Reset Lineup !reset !reset Resets the current lineup
Sign !sign position !sign LW Sign to the current lineup in a particular position
Countersign !sign position playerName !sign LW Thinge Sign another player to the current lineup in a particular position
Unsign !unsign !unsign Unsign yourself from the current lineup
Unsign Globally !unsignglobal !unsignglobal Unsign yourself from all lineups
Counter-unsign !unsign name !unsign Thinge Unsign another player from the current lineup
Clear position !clearpos position !clearpos LW Removes a player by position
Join Subs Bench !sub !sub Joins a lineups sub bench, where you will be automatically inserted into the current lineup if someone unsigns
Leave Subs Bench !unsub !unsub Leaves the lineups sub bench, so you will not replace anyone if they unsign
Matchmaking
Name Command Example Description
View Challenges !challenges !challenges Returns the current teams available to challenge
Challenge Team !challenge teamCode !challenge BB Challenges the specified team, by their team code
Unchallenge Team !unchallenge !unchallenge Unchallenges the team you are currently challenging, abandoning the match
Search For Opposition !search !search Starts an opposition search, where an advert will be sent to other team channels, and your team will appear on the challenges list.
Stop Search !stopsearch !stopsearch Stops the current search, if your team is searching (via !search)
Ready Match !ready server id !ready 12 When the match is ready (a team is set and lineups are full), this starts the match, sending both teams to the specified server. Server ID's are from the !servers command
Re-setup Server !setupserver serverId uniqueMatchId !setupserver 12 If a match has been readied, and the server changes, or just needs to be re-setup, this command will handle it. The match id is available from the previous !ready message.
General
Name Command Example Description
Request Sub !requestsub position serverid !requestsub LW 12 Requests a sub in the current channel for the position specified. Once accepted, the player will be sent the server address to join.
Accept Sub Request !acceptsub token !acceptsub ae7543ba Accepts the sub request advertised by someone using the !requestsub command
Cancel Sub Request !cancelsub token !cancelsub ae7543ba Cancels the sub request advertised by someone using the !requestsub command
Highlight channel !here !here Tells CoachBot to highlight everyone in the channel via the @here mention, in order to get them to sign up
Recent Match List !recentmatches !recentmatches Returns a list of matches recently played by the current channel
Server Administration
Name Command Example Description
Server List !servers N/A Get the full list of available servers for the current region
Enable Single Keeper !enablesinglekeeper server id !enablesinglekeeper 12 Enables single keeper on the specified server
Disable Single Keeper !disablesinglekeeper server id !disablesinglekeeper 12 Disables single keeper on the specified server
View Server Maps !maps server id !maps 12 Return the map list for the specified server
Change Map !changemap serveId  mapName !changemap 12 8v8_vienna Changes the map to the specified map on the specified server
View Server Kits !kits server id !kits 12 Return the kit list for the specified server
Change Kits !changekits server id homeKitId  awayKitId !changekits 12 31 42 Changes the kits to the specified kits on the specified server
Server Info !serverinfo server id !serverinfo 12 Returns granular server info for the specified server
